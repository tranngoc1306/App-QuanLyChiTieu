# App-QuanLyChiTieu
Bài tập môn lập trình .NET1 xây dựng ứng dụng quản lý chi tiêu bằng .Net MAUI

Ứng dụng quản lý chi tiêu EspenseTracker được xây dựng bằng .Net MAUI, hỗ trợ đa nền tảng. Ứng dụng được xây dựng theo kiến trúc MVVM. 

Ứng dụng có các chức năng cơ bản:
1.	Quản lý người dùng:	Đăng ký tài khoản; Đăng nhập hệ thống; Cập nhật thông tin; Đổi mật khẩu; Đăng xuất; Xóa tài khoản
2.	Quản lý giao dịch: Thêm giao dịch (chi phí, thu nhập); Lưu giao dịch; Sửa giao dịch; Xóa giao dịch
3.	Quản lý danh mục: Xem danh mục; Phân loại danh mục (theo chi phí, thu nhập)
4.	Quản lý ví tiền: Xem danh sách ví tiền; Thêm ví mới; Chỉnh sửa thông tin; Xóa ví; Cập nhật số dư; Thiết lập ví mặc định; Thiết lập ngân sách
5.	Báo cáo và thống kê: Xem tổng chi phí; Xem tổng thu nhập; Thống kê theo thời gian; Thống kê theo danh mục; Lọc giao dịch theo ví
6.	Cài đặt hệ thống: Quản lý danh mục mặc định; Quản lý ngôn ngữ; Quản lý tiền tệ


Cài đăt:

• Database: cơ sở dữ liệu cục bộ - SQLite

•	Công cụ phát triển và cài đặt: Visual Studio 2022 (17.8 trở lên), .NET 8.0 SDK (tối thiểu)
